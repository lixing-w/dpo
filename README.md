# dpo
Replicating DPO paper with LoRA fine-tune. Final Project for NLP Spring 26

Using Qwen2.5-0.5B Model and HuggingFaceH4/ultrafeedback_binarized Dataset.
Full-funetune for SFT and LoRA for DPO.

1. SFT: sft.ipynb
2. Reward model for held-out data: reward.ipynb
3. DPO fine-tune: dpo.ipynb

Note: The reward model is just used to check if DPO fine-tuning is working.
